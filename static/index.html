General Routing

Route HTTP Method Description
/ GET Display the home Page

Route Information for users

Routes HTTP Method Description
/users/ GET Retrieve all users from the database
/users/:id GET Retrieve a specific user
/users/register post Register a new user
/users/update/:id patch Edit user information
/users/delete/:id delete Delete a specific user
/users/login post Allow user to log in

Route Information for Products

Routes HTTP Method Description
/products/ get Retrieve all products
/products/:id get Retrieve a specific product
/products/add post Add a new product
/products/update/:id patch Edit a specific product
/products/delete/:id patch delete a specific product

Route Information for Cart

Routes HTTP Method Description
/cart/ get Retrieve all products in cart
/cart/delete/:id delete Delete a specific product in cart
/cart/delete delete Clear the cart
/cart/add/ post Add a product to the cart
